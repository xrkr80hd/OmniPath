# Session Zero Media

Drop Session Zero ambience assets here.

## Folders

- `video/`
  Place the looping portal background video here.
- `audio/`
  Place the Session Zero background music here.

## Recommended filenames

- `video/portal-loop.mp4`
- `audio/session-zero-theme.mp3`

These names are only conventions for now. The app is not wired to them yet.
